About This Model

This is a simplified version of a project management model based on Vensim system dynamics principles. The model includes:

  • Work Accomplished - A stock that accumulates completed work
  • Work Remaining - Calculated as Initial Project Definition minus Work Accomplished
  • Work Flow - The rate at which work is completed (stops when project is done)
  • Project Is Done - A condition that becomes true when work accomplished reaches the initial definition

The model demonstrates key system dynamics concepts like stocks, flows, and feedback loops.
